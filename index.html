Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible conten shown to the user.
  • Heading tag lines must be used in order from H1 to H6 with the biggest heading starting with H1.
  • Alt text improves search engine rankings for the website.

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element.
  • There are three styles of CSS Inline CSS, Internal CSS Style Sheet, External CSS Style Sheet.
  • CSS defines colors, aesthetics, visual layout and organization.
  • CSS is short for Cascading Style Sheets

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git push origin pushes changes we have locally to remote branch
